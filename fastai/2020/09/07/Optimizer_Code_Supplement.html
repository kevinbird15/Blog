Introduction

This post is meant to be a supporting guide that can be used with the incredibly detailed post from Sebastian Ruder. The goal is to implement each of these optimizers using fastai and run them on a small problem.

The sample problem that will be used is the one created by Jeremy and Rachel in the graddesc spreadsheet.

Because this post is generated using Jupyter Notebooks, you can also interact with any of these concepts and modify them as you see fit. In as many places as possible, I will try to connect the code back to the concepts by utilizing the greek letters Sebastian (and usually the paper) uses

a_real = 2
b_real = 30

a_real is a weight and b_real is a bias they form the equation $a\_real*x + b\_real= y$
Both a and b are parameters and will be learned by our optimizer through the loss function.
In a real world problem, these values would be unknown and there would typically be a lot more weights, but the concept will remain the same whether there are 2 parameters or 175 billion.

Gradient Descent Variants

Batch Gradient Descent

Forumula:
$\theta = \theta - \eta \cdot \nabla_\theta J( \theta)$

Code:

def sgd_step(p, lr, **kwargs):
    p.data.add_(p.grad.data, alpha=-lr)

Translation:
$\theta$ = p
$\eta$ = lr
$\nabla_\theta J( \theta)$ = p.grad.data

def _do_epoch_BGD(self):
    self._do_epoch_train()
    self._step()
    self('after_step')
    self.opt.zero_grad()
    self._do_epoch_validate()

def _do_one_batch_BGD(self):
    self.pred = self.model(*self.xb)
    self('after_pred')
    if len(self.yb): self.loss = self.loss_func(self.pred, *self.yb)
    self('after_loss')
    if not self.training or not len(self.yb): return
    self('before_backward')
    self._backward()
    self('after_backward')

Stochastic Gradient Descent

Formula:
$\theta = \theta - \eta \cdot \nabla_\theta J( \theta; x^{(i)}; y^{(i)})$

Code:

def sgd_step(p, lr, **kwargs):
    p.data.add_(p.grad.data, alpha=-lr)

Translation:
$\theta$ = p
$\eta$ = lr
$\nabla_\theta J( \theta; x^{(i)}; y^{(i)})$ = p.grad.data

def _do_epoch_SGD(self):
    self._do_epoch_train()
    self._do_epoch_validate()

def _do_one_batch_SGD(self):
    #pdb.set_trace()
    for i,b in enumerate(zip(self.xb[0][None].T,self.yb[0][None].T)):
        x,y = b
        self.pred = self.model(x)
        self('after_pred')
        if len(y): self.loss = self.loss_func(self.pred, y)
        self('after_loss')
        if not self.training or not len(self.yb): return
        self('before_backward')
        self._backward()
        self('after_backward')
        self._step()
        self('after_step')
        self.opt.zero_grad()

Beyond Vanilla Gradient Descent

SGD with Momentum

$ \begin{align} \begin{split} v_t &= \gamma v_{t-1} + \eta \nabla_\theta J( \theta) \\ \theta &= \theta - v_t \end{split} \end{align} $

Nesterov Accelerated Gradient

$ \begin{align} \begin{split} v_t &= \gamma v_{t-1} + \eta \nabla_\theta J( \theta - \gamma v_{t-1} ) \\ \theta &= \theta - v_t \end{split} \end{align} $

Adagrad

$g_{t, i} = \nabla_\theta J( \theta_{t, i} )$
$\theta_{t+1, i} = \theta_{t, i} - \eta \cdot g_{t, i}$
$\theta_{t+1, i} = \theta_{t, i} - \dfrac{\eta}{\sqrt{G_{t, ii} + \epsilon}} \cdot g_{t, i}$
$\theta_{t+1} = \theta_{t} - \dfrac{\eta}{\sqrt{G_{t} + \epsilon}} \odot g_{t}$

Adadelta

$E[g^2]_t = \gamma E[g^2]_{t-1} + (1 - \gamma) g^2_t$
$ \begin{align} \begin{split} \Delta \theta_t &= - \eta \cdot g_{t, i} \\ \theta_{t+1} &= \theta_t + \Delta \theta_t \end{split} \end{align} $
$\Delta \theta_t = - \dfrac{\eta}{\sqrt{G_{t} + \epsilon}} \odot g_{t}$
$\Delta \theta_t = - \dfrac{\eta}{\sqrt{E[g^2]_t + \epsilon}} g_{t}$
$\Delta \theta_t = - \dfrac{\eta}{RMS[g]_{t}} g_t$
$E[\Delta \theta^2]_t = \gamma E[\Delta \theta^2]_{t-1} + (1 - \gamma) \Delta \theta^2_t$
$RMS[\Delta \theta]_{t} = \sqrt{E[\Delta \theta^2]_t + \epsilon}$
$ \begin{align} \begin{split} \Delta \theta_t &= - \dfrac{RMS[\Delta \theta]_{t-1}}{RMS[g]_{t}} g_{t} \\ \theta_{t+1} &= \theta_t + \Delta \theta_t \end{split} \end{align} $
